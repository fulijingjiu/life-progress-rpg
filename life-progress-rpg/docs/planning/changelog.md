## [0.1.0] - 2026-XX-XX

### 首次发布

这是人生进度条 RPG 的第一个版本，专注于验证核心价值主张。

### 新增

#### 核心功能
- 🌱 **人生进度条** — 可视化人生进度、等级、章节
- ✍️ **每日记录** — 心情、能量、内容、标签记录
- 🤖 **AI 即时洞察** — 记录后 AI 简短反馈
- 📖 **章节系统** — 7 个人生章节划分

#### 用户体验
- 🎨 **首次引导** — 3 步快速创建角色
- 🎭 **5 种主题** — 代表不同人生阶段
- ✨ **进度动画** — 数字滚动效果
- 📱 **响应式设计** — 适配移动端

#### 数据管理
- 💾 **本地存储** — IndexedDB 数据持久化
- 📊 **记录历史** — 查看历史记录
- ⚙️ **基础设置** — 修改昵称、头像、主题

### 技术特性

- React 18 + TypeScript
- Tailwind CSS
- Framer Motion 动画
- Zustand 状态管理
- IndexedDB (Dexie.js)
