# 产品路线图

> 人生进度条 RPG 开发计划

## 版本规划

```
v0.1 MVP          → 基础进度条 + 记录 + AI 总结
v0.2 成就系统      → 徽章 + 经验值 + 升级
v0.3 分享功能      → 卡片生成 + 社交分享
v1.0 正式发布      → 完整功能 + 优化体验
v1.x 社交功能      → 排行榜 + 关注 + 互动
v2.0 深度 AI      → 多轮对话 + 人生规划
```

---

## v0.1 MVP

**目标**：验证核心价值主张

### 交付时间
- 周期：1 周
- 日期：2026-XX-XX

### 核心功能

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 人生进度条 | 显示人生进度、等级、章节 | P0 |
| 每日记录 | 心情 + 能量 + 内容 + 标签 | P0 |
| AI 即时洞察 | 记录后 AI 给出简短反馈 | P0 |
| 基础设置 | 昵称、头像、主题 | P0 |
| 本地存储 | IndexedDB 数据持久化 | P0 |

### 用户旅程

```
注册 → 输入生日 → 看到进度条 → 记录今天 → 收到 AI 反馈 → 返回首页
```

### 成功指标

| 指标 | 目标 |
|------|------|
| 完成注册率 | > 60% |
| 首次记录率 | > 40% |
| 次日留存 | > 30% |
| 7 日留存 | > 15% |

---

## v0.2 成就系统

**目标**：增加游戏化粘性

### 交付时间
- 周期：3 天
- 日期：v0.1 + 3 天

### 新增功能

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 经验值系统 | 记录获取 XP，积累升级 | P0 |
| 成就徽章 | 预设 10 个成就 | P0 |
| 连续记录 | 记录连续天数 | P0 |
| 徽章动画 | 解锁时 3D 翻转 | P1 |
| 章节进度 | 当前章节进度条 | P0 |

### 成就列表

| 成就 | 条件 |
|------|------|
| 🌱 萌芽者 | 完成首次记录 |
| 🔥 连续 7 天 | 连续记录 7 天 |
| 💎 洞察者 | 首次使用 AI |
| 📊 数据达人 | 记录满 30 天 |

### 成功指标

| 指标 | 目标 |
|------|------|
| 成就解锁率 | > 50% |
| 连续记录完成率 | > 30% |
| 分享率 | > 10% |

## v0.3 分享功能

**目标**：实现社交传播

### 交付时间
- 周期：3 天
- 日期：v0.2 + 3 天

### 新增功能

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 分享卡片 | html2canvas 生成图片 | P0 |
| 成就卡片 | 徽章 + AI 贺词 | P0 |
| 日常卡片 | 进度 + 心情 + AI | P1 |
| 多主题卡片 | 5 种卡片模板 | P1 |
| 二维码 | 扫码进入 App | P0 |

### 卡片类型

| 类型 | 内容 | 触发 |
|------|------|------|
| 日常进度卡 | 进度条 + 等级 + AI 一句话 | 每日记录后 |
| 成就卡 | 徽章 + 解锁条件 + AI 贺词 | 成就解锁时 |

### 成功指标

| 指标 | 目标 |
|------|------|
| 分享率 | > 15% |
| 卡片生成成功率 | > 95% |
| 社交回流率 | > 5% |
