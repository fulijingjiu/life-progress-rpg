## 设计原则

### 核心理念

> **爽感让你打开，价值让你留下**

所有设计决策都围绕这两个维度展开：
- **爽感层** — 进度条动画、徽章解锁、分享卡片
- **价值层** — AI 洞察、趋势分析、人生建议

### 设计目标

| 目标 | 指标 |
|------|------|
| 首次打开 | 3 分钟内看到人生进度条 |
| 次日留存 | > 40% |
| 7 日留存 | > 25% |
| 分享率 | > 15% |
