## 一、产品定位

### 1.1 一句话定义
把人生变成一场 RPG 游戏——你每天的记录都是经验值，AI 是你的专属人生向导。

### 1.2 用户画像

| 类型 | 特征 | 核心诉求 |
|------|------|---------|
| 焦虑型 | 25-35岁，工作压力大，总觉得时间不够用 | 看到"进度"获得掌控感 |
| 记录型 | 喜欢写日记、打卡、做手账 | 有一个漂亮的载体记录生活 |
| 社交型 | 喜欢发朋友圈、小红书 | 生成好看的分享卡片 |
| 探索型 | 对自我成长、心理学感兴趣 | AI 分析给出洞察 |

### 1.3 核心情绪曲线

```
好奇（首次打开）
   ↓
惊喜（看到自己的人生进度条动起来）
   ↓
投入（开始记录第一天）
   ↓
期待（每天打开看 AI 说了什么）
   ↓
成就（解锁徽章、进入新章节）
   ↓
分享（生成卡片发朋友圈）
   ↓
依赖（习惯每天记录，不看不舒服）
```

## 二、用户旅程地图

### 2.1 首次打开（Onboarding）

**目标**：3 分钟内让用户看到自己的人生进度条，产生"哇"的感觉。

| 步骤 | 界面 | 用户行为 | 系统反馈 | 爽点设计 |
|------|------|---------|---------|---------|
| 1 | 启动页 | 打开 App | 显示像素风 Logo + 加载动画 | 8-bit 音效 + 进度条填充 |
| 2 | 欢迎页 | 点击"开始游戏" | 全屏文字逐字打出："欢迎来到你的人生 RPG" | 打字机效果，营造仪式感 |
| 3 | 创建角色 | 输入昵称、选择生日 | 实时计算并显示："你已活过 X 天" | 数字从 0 滚动到真实值，震撼 |
| 4 | 选择主题 | 点击 5 种配色方案 | 预览实时切换 | 每种主题配不同 BGM 预览 |
| 5 | 生成面板 | 点击"生成我的角色" | 播放 3 秒生成动画，然后展示完整面板 | **核心爽点：人生进度条第一次出现** |

**Onboarding 文案**：

> "每个人的人生都是一场 RPG。
> 你已经在地球上度过了 11,361 天。
> 当前等级：Lv.31
> 当前章节：第4章「扎根」
> 进度：38.88%
> 
> 准备好升级了吗？"

---

### 2.2 日常使用（Daily Loop）

**目标**：让用户每天愿意花 2 分钟打开 App，记录 + 看 AI 反馈。

#### 场景 A：早晨打开（查看今日状态）

| 元素 | 设计 |
|------|------|
| 打开动画 | 从锁屏通知点击进入，App 启动时进度条从昨日位置平滑填充到今日 |
| 首页状态 | 顶部显示"今天是你在地球的第 X 天"，数字带呼吸动画 |
| 今日提示 | 根据当前章节，显示一句今日金句（如扎根期："今天种下的根，明天会长成树"） |
| 快捷入口 | 底部大按钮"记录今天"，带脉冲动画吸引点击 |

#### 场景 B：记录今天（核心交互）

**记录流程**：

```
点击"记录今天" → 弹出底部Sheet
   ↓
【心情选择器】
  一排 5 个表情：😫 😐 🙂 😊 🤩
  选中后放大 + 轻微弹跳
   ↓
【能量滑块】
  0-10 滑动，滑块是小火苗图标
  滑动时背景色从灰变橙变红
   ↓
【今日一句话】
  输入框占位符："今天发生了什么？"
  可选：语音输入（按住说话）
   ↓
【分类标签】
  横向滑动选择：💼工作 📚学习 🏃健康 💕关系 🎮娱乐
  可多选，选中后标签发光
   ↓
【提交】
  按钮文案："存档今日"
  点击后：按钮变成加载 → 成功动画（✅弹出 + 轻微震动）
   ↓
【AI 反馈】
  1-2 秒后，底部弹出 AI 气泡：
  "今天能量值 7/10，心情不错！
   你提到了'项目上线'，这是本周第 3 次提到工作，
   记得给自己留点放松时间 🌿"
```

**记录完成后的奖励动画**：
- 屏幕中央弹出"+1 经验值"
- 经验条闪烁填充一小段
- 如果连续记录，显示"🔥 连续 X 天"火焰动画

#### 场景 C：查看 AI 对话（深度互动）

**入口**：首页底部"与人生向导对话"按钮

## 四、动效与微交互设计

### 4.1 核心动效清单

| 场景 | 动效 | 时长 | 目的 |
|------|------|------|------|
| 启动 | Logo 像素化出现 + 8-bit 音效 | 1.5s | 建立游戏氛围 |
| 进度条填充 | 从 0 平滑滚动到当前值 | 2s | 震撼感，数字滚动比直接显示更有冲击力 |
| 记录提交 | 按钮 → 加载 → ✅ 弹出 + 震动 | 1s | 正向反馈 |
| 徽章解锁 | 徽章从远处飞入 + 3D 翻转 | 2s | 成就感 |
| 章节切换 | 翻书/卷轴展开效果 | 3s | 仪式感 |
| AI 回复 | 流式打字 + 光标闪烁 | 实时 | 拟人感 |
| 连续记录 | 火焰图标跳动 + 数字递增 | 0.5s | 正向强化 |
| 下拉刷新 | 进度条重新播放填充 | 1.5s | 趣味性 |

### 4.2 音效设计

| 场景 | 音效 | 风格 |
|------|------|------|
| 启动 | 8-bit 开机音 | 复古游戏 |
| 点击按钮 | 轻微"滴"声 | 清脆 |
| 记录提交 | 升级音 | 正向反馈 |
| 徽章解锁 | 胜利号角 | 成就感 |
| 章节切换 | 翻书声 + 风铃声 | 仪式感 |
| 连续记录 | 火焰"呼"声 | 力量感 |
| AI 回复 | 轻微打字机声 | 拟人 |

**注意**：所有音效默认关闭，用户可在设置中开启。

## 五、竞品差异化总结

| 维度 | 现有工具 | 我们的设计 |
|------|---------|-----------|
| 情感基调 | 焦虑（死亡倒计时） | 爽感（RPG 升级） |
| 交互深度 | 静态展示 | 动态游戏化 + AI 对话 |
| 社交传播 | 无 | 精美卡片 + 病毒式文案 |
| 用户粘性 | 一次性查看 | 每日记录习惯养成 |
| 数据价值 | 无 | AI 洞察 + 人生趋势分析 |
| 个性化 | 无 | 章节系统 + AI 个性化反馈 |
