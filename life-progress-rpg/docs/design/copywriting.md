# 文案风格指南

> 语气、用词、模板

## 文案原则

### 基本原则

1. **简洁有力** — 少即是多
2. **游戏感** — RPG 风格但不中二
3. **温暖洞察** — AI 像朋友而非机器
4. **行动导向** — 引导用户下一步

### 语气规范

| 场景 | 语气 | 说明 |
|------|------|------|
| 系统文案 | 简洁有力 | 游戏化但不刻意 |
| AI 文案 | 温暖洞察 | 像朋友，有洞察力 |
| 成就文案 | 夸张有仪式感 | 值得截图 |
| 引导文案 | 亲切鼓励 | 不强迫，自然引导 |

## 关键文案模板

### 启动页

```
每个人的人生都是一场 RPG。

你已经在地球上度过了 {days} 天。
当前等级：Lv.{age}
当前章节：第{chapter}章「{chapterName}」
进度：{progress}%

🤖 AI 人生向导：
"{aiInsight}"

准备好升级了吗？
```

### 每日金句（按章节）

| 章节 | 金句 |
|------|------|
| 🌱 萌芽 | "每一天都是新的技能点" |
| 📚 求学 | "知识是你最早的装备" |
| 🔍 探索 | "世界很大，但你正在变大" |
| 🌿 扎根 | "今天种下的根，明天会长成树" |
| 🌸 绽放 | "你的影响力正在指数级增长" |
| 🍂 沉淀 | "智慧是你留给世界的遗产" |
| 🌅 归途 | "每一刻都值得被记住" |

### 心情文案

| 心情 | 描述 |
|------|------|
| 😫 | "今天有点艰难，但你坚持下来了" |
| 😐 | "平静也是一种力量" |
| 🙂 | "普通的一天，也有它的意义" |
| 😊 | "心情不错！继续保持" |
| 🤩 | "太棒了！这是值得纪念的一天" |

### 能量文案

| 能量 | 描述 |
|------|------|
| 0-2 | "需要好好休息了" |
| 3-4 | "有点疲惫，注意调整" |
| 5-6 | "状态一般，可以完成日常任务" |
| 7-8 | "状态不错！可以挑战更高目标" |
| 9-10 | "满血复活！今天属于你" |

## AI 文案模板

### 日常问候

```
早上好，{nickname}。
今天是你在地球的第 {days} 天。
本周你的平均心情是 {mood}，{trend}上周 {change}！
```

### 记录分析

```
今天能量值 {energy}/10，心情 {emoji}！

你提到了'{keyword}'，这是本周第 {count} 次提到 {category}。

📊 数据洞察：
{insight}

💡 AI 建议：
{suggestion}
```

### 疲劳提醒

```
我注意到你这周有 {low_energy_days} 天能量值低于 5。
而且你连续 {no_social_days} 天没有选择'关系'标签了。

要不要给自己放个假？
或者约朋友吃顿饭？

{history_example}
```

### 成就贺词

```
🎉 恭喜你解锁【{achievement}】！

你已经超越了 {percent}% 的用户。

📊 数据洞察：
{insight}

💡 继续前进：
{next_step}
```

### 周报开场

```
📊 本周数据报告

{date_range}

心情趋势：{mood_trend}
能量趋势：{energy_trend}
最高心情：{peak_day} ({peak_value})
最低心情：{low_day} ({low_value})

📈 关键发现：
{findings}

💡 下周建议：
{suggestions}
```

### 章节切换

```
🎉 恭喜进入第{chapter}章「{chapterName}」！

{chapter_description}

📊 数据预测：
{prediction}

🎯 本章目标建议：
{goals}

{ai_message}
```

## 按钮文案

| 场景 | 文案 | 风格 |
|------|------|------|
| 主按钮 | 存档今日 | 游戏化 |
| 次按钮 | 继续游戏 | 游戏化 |
| AI 对话 | 与人生向导对话 | 神秘感 |
| 分享 | 生成分享卡片 | 直接 |
| 查看数据 | 查看数据洞察 | 价值导向 |

### 不推荐的按钮文案

| ❌ 不推荐 | ✅ 推荐 | 原因 |
|----------|--------|------|
| 提交 | 存档今日 | 游戏化 |
| 发送 | 发送（保留） | AI 对话场景下可以 |
| 查看更多 | 深入了解 | 更吸引 |
| 设置 | 调整设定 | 游戏化 |

## 错误文案

| 场景 | 文案 |
|------|------|
| 网络断开 | "信号丢失...但你的进度已本地保存" |
| AI 响应超时 | "人生向导正在思考，请稍等..." |
| 超时后 | "向导暂时离线，你的记录已保存" |
| 提交失败 | "存档失败，但别担心，数据已本地备份" |
| 截图失败 | "卡片生成中...如果失败，可以手动截图" |
