# 文案风格指南

> 语气、用词、模板 | 版本：v2.0 | 日期：2026-07-23

---

## 文案原则

### 基本原则

1. **简洁有力** — 少即是多
2. **游戏感** — RPG 风格但不中二
3. **温暖洞察** — AI 像朋友而非机器
4. **行动导向** — 引导用户下一步
5. **正向激励** — 永远给予希望

### 语气规范

| 场景 | 语气 | 说明 |
|------|------|------|
| 系统文案 | 简洁有力 | 游戏化但不刻意 |
| AI 文案 | 温暖洞察 | 像朋友，有洞察力 |
| 成就文案 | 夸张有仪式感 | 值得截图 |
| 引导文案 | 亲切鼓励 | 不强迫，自然引导 |
| 错误文案 | 轻松化解 | 不让用户感到挫败 |
| 提醒文案 | 关心而非催促 | 让人想行动 |

### 禁用词

| 禁用 | 替代 |
|------|------|
| 死亡 | 人生终点、到站 |
| 死 | 到站、终点 |
| 老了 | 进入新阶段、成熟 |
| 剩余时间 | 剩余旅程、剩余章节 |
| 倒计时 | 进度、前进 |
| 焦虑 | 期待 |
| 压力 | 挑战 |
| 失败 | 成长中 |
| 错误 | 新发现 |

## AI 文案模板

### 日常问候

```
早上好，{nickname}。
今天是你在地球的第 {days} 天。
本周你的平均心情是 {mood}，{trend}上周 {change}！
```

### 记录分析

```
今天能量值 {energy}/10，心情 {emoji}！

你提到了'{keyword}'，这是本周第 {count} 次提到 {category}。

📊 数据洞察：
{insight}

💡 AI 建议：
{suggestion}
```

### 疲劳提醒

```
我注意到你这周有 {low_energy_days} 天能量值低于 5。
而且你连续 {no_social_days} 天没有选择'关系'标签了。

要不要给自己放个假？
或者约朋友吃顿饭？

{history_example}
```

### 成就贺词

```
🎉 恭喜你解锁【{achievement}】！

你已经超越了 {percent}% 的用户。

📊 数据洞察：
{insight}

💡 继续前进：
{next_step}
```

### 周报开场

```
📊 本周数据报告

{date_range}

心情趋势：{mood_trend}
能量趋势：{energy_trend}
最高心情：{peak_day} ({peak_value})
最低心情：{low_day} ({low_value})

📈 关键发现：
{findings}

💡 下周建议：
{suggestions}
```

### 章节切换

```
🎉 恭喜进入第{chapter}章「{chapterName}」！

{chapter_description}

📊 数据预测：
{prediction}

🎯 本章目标建议：
{goals}

{ai_message}
```

### AI 主动关怀

```
{nickname}，我注意到...

你最近 {observation}。

这让我想起 {past_pattern}。

要不要 {suggestion}？

{encouragement}
```

### 里程碑达成

```
🎯 里程碑达成！

你已经连续记录 {days} 天！

{stats_summary}

{ai_commentary}

{next_milestone_preview}
```

## 按钮文案

| 场景 | 文案 | 风格 |
|------|------|------|
| 主按钮 | 存档今日 | 游戏化 |
| 次按钮 | 继续游戏 | 游戏化 |
| AI 对话 | 与人生向导对话 | 神秘感 |
| 分享 | 生成分享卡片 | 直接 |
| 查看数据 | 查看数据洞察 | 价值导向 |
| 确认 | 出发吧 | 鼓励 |
| 取消 | 稍后再说 | 轻松 |
| 重试 | 再试一次 | 积极 |

### 不推荐的按钮文案

| ❌ 不推荐 | ✅ 推荐 | 原因 |
|----------|--------|------|
| 提交 | 存档今日 | 游戏化 |
| 发送 | 发送（保留） | AI 对话场景下可以 |
| 查看更多 | 深入了解 | 更吸引 |
| 设置 | 调整设定 | 游戏化 |
| 关闭 | 稍后再说 | 轻松 |
| 确定 | 好的 | 自然 |
| 删除 | 移除 | 温和 |

## 错误文案

| 场景 | 文案 |
|------|------|
| 网络断开 | "信号丢失...但你的进度已本地保存" |
| AI 响应超时 | "人生向导正在思考，请稍等..." |
| 超时后 | "向导暂时离线，你的记录已保存" |
| 提交失败 | "存档失败，但别担心，数据已本地备份" |
| 截图失败 | "卡片生成中...如果失败，可以手动截图" |
| 登录过期 | "登录状态需要刷新，重新连接中..." |
| 数据加载失败 | "数据读取失败，不过你的记录都在" |
