# 人生进度条 RPG — 用户体验设计文档 v2.0 (UX Spec)

> 版本：v2.0 | 日期：2026-07-23
> 核心原则：**爽感让你打开，价值让你留下**

---

## 一、产品定位

### 1.1 一句话定义
把人生变成一场 RPG 游戏——爽感驱动打开，价值驱动留存，AI 让每一天都有意义。

### 1.2 爽感 vs 价值 双轮驱动

| 维度 | 爽感层 | 价值层 |
|------|--------|--------|
| 驱动力 | 多巴胺、好奇心、社交炫耀 | 自我成长、掌控感、人生意义 |
| 表现形式 | 进度条动画、徽章解锁、分享卡片 | AI 洞察报告、趋势分析、人生建议 |
| 用户感受 | "好酷！" "我要发朋友圈" | "原来我是这样的" "我知道该怎么做了" |
| 留存作用 | 短期吸引 | 长期依赖 |

---

## 二、AI 价值层级

| 层级 | 能力 | 示例 | 价值感 |
|------|------|------|--------|
| L1 记录总结 | 总结当日记录 | "今天能量值 7，心情不错" | ⭐⭐ |
| L2 趋势分析 | 对比本周 vs 上周 | "本周心情比上周提升 0.5" | ⭐⭐⭐ |
| L3 关联发现 | 发现行为模式 | "能量值低的日子 87% 与加班相关" | ⭐⭐⭐⭐ |
| L4 预测建议 | 基于历史预测未来 | "下周三如果完成运动，能量值预计提升 23%" | ⭐⭐⭐⭐⭐ |
| L5 人生规划 | 长期人生建议 | "基于你的数据，建议 35 岁前完成..." | ⭐⭐⭐⭐⭐ |

---

## 三、价值体系设计

### 3.1 价值输出矩阵

| 时间维度 | 价值内容 | 输出形式 |
|---------|---------|---------|
| 每日 | AI 当日洞察 | 记录提交后自动弹出 |
| 每周 | 周报：心情/能量趋势 + 行为模式 + 下周建议 | 周日推送 + 可生成卡片 |
| 每月 | 月报：人生数据全景 + 成长轨迹 + 目标达成 | 月度里程碑时推送 |
| 每季 | 季报：季度主题总结 + 人生阶段评估 | 章节切换时推送 |
| 每年 | 年度史诗：全年关键词 + 人生转折点 + 新年建议 | 年底自动生成 |

---

## 四、用户成长路径

```
Day 1：好奇打开 → 看到进度条 → "哇，好酷"（爽感）
   ↓
Day 2-7：每天记录 → 看到 AI 反馈 → "原来我是这样的"（价值初现）
   ↓
Day 8-30：连续记录 → 收到第一份周报 → "我的数据居然这么有意思"（价值确认）
   ↓
Day 31-90：习惯养成 → 收到月报 + 人生建议 → "AI 比我自己还懂我"（价值深化）
   ↓
Day 91+：深度依赖 → 重大决策咨询 AI → "没有它我不知道怎么决定"（价值不可替代）
```

---

## 五、爽感与价值的平衡设计

### 5.1 交互节奏设计

```
用户操作 → 爽感即时反馈（0-1秒）→ 价值延迟呈现（1-3秒）
   ↓
例：点击"存档今日"
   ↓
0ms：按钮按下动画（爽感）
500ms：✅弹出 + 震动 + +1经验值（爽感）
1000ms：🔥连续天数更新（爽感）
1500ms：AI 洞察气泡滑入（价值）
2000ms：数据图表展开（价值）
```

### 5.2 文案双轨设计

| 场景 | 爽感文案 | 价值文案 |
|------|---------|---------|
| 记录完成 | "+1 经验值！存档成功！" | "你的数据又丰富了一点，AI 将给出更精准的建议" |
| 徽章解锁 | "🏆 解锁新成就！超越 73% 用户！" | "连续记录 7 天，你的平均心情提升了 0.8 分" |
| 章节切换 | "🎉 进入新章节！等级提升！" | "基于你的数据，新章节建议重点关注..." |
| 每日打开 | "今天是你在地球的第 X 天！" | "本周目标完成度 67%，再坚持 2 天解锁..." |

## 六、竞品差异化（爽感 + 价值）

| 维度 | 现有工具 | 人生进度条 RPG |
|------|---------|-----------|
| 情感基调 | 焦虑（死亡倒计时） | **爽感 + 成长（RPG 升级）** |
| 交互深度 | 静态展示 | **动态游戏化 + 深度 AI 对话** |
| 社交传播 | 无 | **价值型分享（展示成长而非炫耀）** |
| 用户粘性 | 一次性查看 | **习惯养成 + 人生依赖** |
| 数据价值 | 无 | **AI 洞察 + 人生规划 + 决策支持** |
| 个性化 | 无 | **章节系统 + 个人数据驱动的定制化** |
| 长期价值 | 无 | **人生档案 + 年度史诗 + 成长轨迹** |
