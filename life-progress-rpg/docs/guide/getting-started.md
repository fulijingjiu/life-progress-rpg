# 快速开始

> 5 分钟快速上手人生进度条 RPG

## 环境要求

| 要求 | 版本 | 说明 |
|------|------|------|
| Node.js | ≥ 18.0 | 前端运行环境 |
| npm | ≥ 9.0 | 包管理器 |
| 浏览器 | Chrome 110+ / Safari 16+ / Firefox 120+ | 现代浏览器 |

## 安装步骤

### 1. 克隆项目

```bash
git clone https://github.com/yourusername/life-progress-rpg.git
cd life-progress-rpg
```

### 2. 安装依赖

```bash
npm install
```

### 3. 配置环境变量

创建 `.env.local` 文件：

```bash
# AI 服务配置
VITE_AI_PROVIDER=openai
VITE_AI_API_KEY=your_api_key

# 数据库配置（可选，开发环境使用本地存储）
VITE_USE_LOCAL_STORAGE=true
```

### 4. 启动开发服务器

```bash
npm run dev
```

访问 http://localhost:5173 即可看到应用。

## 首次使用引导

### 第一步：创建角色

首次打开应用，你会看到欢迎页。点击「开始游戏」进入角色创建：

1. **输入昵称** — 你在游戏中的名字
2. **选择生日** — 用于计算人生进度（仅年份，隐私保护）
3. **选择主题** — 5 种配色方案，代表不同人生阶段

### 第二步：看到你的进度

创建角色后，系统会计算并展示：

```
🤖 欢迎，张三！
你已经在地球上度过了 11,361 天。
当前等级：Lv.31
当前章节：第4章「扎根」
进度：38.88%

📊 AI 人生向导：
"你正处于人生第4章的早期，
这是建立根基的关键期。
过去 30 天，你 67% 的记录与'工作'相关，
建议下周给'关系'和'健康'多留一些时间。"
```

### 第三步：记录今天

点击首页「记录今天」按钮，进入记录流程：

1. **选择心情** — 从 😫 到 🤩 五个等级
2. **滑动能量** — 0-10 的能量值
3. **输入内容** — 今天发生的事情
4. **选择标签** — 工作/学习/健康/关系/娱乐
5. **提交存档** — 点击「存档今日」

提交后，AI 会立即给出今日洞察。

## 基础操作

### 每日使用流程

```
早晨打开 → 查看今日状态 → 记录今天 → 查看 AI 反馈 → 分享卡片
```

### 查看人生数据

点击底部导航「数据」，可以查看：
- 心情趋势图（近 90 天）
- 能量分布图
- 标签时间占比
- 关键发现

### 与 AI 对话

点击「与人生向导对话」，可以：
- 询问最近的状态
- 获取个性化建议
- 制定计划
- 倾诉心事
