# 用户画像

> 目标用户特征与需求分析

## 一、目标用户概述

人生进度条 RPG 的目标用户是 **25-45 岁的城市白领**，他们关心自我成长，对时间有焦虑感，愿意用工具来记录和分析自己的生活。

## 三、用户类型对比

| 类型 | 核心诉求 | 爽点 | 我们的价值 |
|------|----------|------|------------|
| 焦虑探索者 | 掌控感 | 看到进度 | 成长视角替代焦虑视角 |
| 记录爱好者 | 美观载体 | 漂亮界面 | 游戏化 + AI 分析 |
| 社交分享者 | 好素材 | 分享卡片 | 精美卡片 + 成长展示 |
| 自我探索者 | 深度洞察 | AI 分析 | 深度对话 + 人生规划 |

## 四、用户旅程

### 4.1 新用户旅程

```
认知 → 好奇 → 首次打开 → 创建角色 → 首次记录 → 每日习惯
  ↓
渠道：朋友推荐 / 朋友圈分享 / 应用商店 / 社交媒体
```

### 4.2 核心旅程

```
早晨打开 → 查看今日状态 → 记录今天 → 查看 AI 反馈 → 分享/关闭
```

### 4.3 留存旅程

```
Day 1：哇，好酷！进度条好好看
Day 7：连续记录了一周，AI 说得挺准
Day 30：已经成为习惯了，每天不看不舒服
Day 90：有时候会问 AI 人生建议，感觉离不开
```

## 五、用户反馈收集

### 5.1 访谈问题

1. 你平时记录生活吗？用什么方式？
2. 你对"人生进度条"这个概念有什么感受？
3. 什么样的反馈会让你觉得"这个 App 有用"？
4. 你会愿意分享这类内容到朋友圈吗？
5. 你对 AI 给你的人生建议有什么期待？

### 5.2 满意度指标

| 指标 | 目标 | 测量方式 |
|------|------|----------|
| 首次体验满意度 | > 4.5/5 | 首次引导后评分 |
| 功能使用率 | > 60% | 埋点数据 |
| NPS 评分 | > 40 | 问卷调查 |
| 卸载率 | < 10% | 7 日卸载 |
